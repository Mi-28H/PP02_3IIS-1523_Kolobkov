## 🎯 Описание проекта

Проект представляет собой комплексное решение для экзаменационного задания, включающее:

- 🌐 **Embedded-разработку**: программирование ESP8266/ESP32 в режиме точки доступа с веб-интерфейсом управления встроенным светодиодом.

## 🔌 Задание №1: Веб-сервер на ESP в режиме точки доступа

### 🔧 Функционал

- Создание точки доступа **SmartHome_Gateway** с паролем **12345678**.
- Запуск простого веб-сервера на порту 80.
- Две управляющие команды:
  - `/on` – включить встроенный светодиод.
  - `/off` – выключить встроенный светодиод.
- Главная страница содержит две кнопки для удобного управления.

### 💻 Листинг кода (основная часть)

```cpp
#include <ESP8266WiFi.h>  // Для ESP8266
// #include <WiFi.h>      // Для ESP32

const char* ssid = "SmartHome_Gateway";
const char* password = "12345678";
WiFiServer server(80);
const int ledPin = LED_BUILTIN;  // GPIO2

void setup() {
  Serial.begin(115200);
  pinMode(ledPin, OUTPUT);
  digitalWrite(ledPin, HIGH);
  WiFi.softAP(ssid, password);
  server.begin();
}

void loop() {
  WiFiClient client = server.available();
  if (!client) return;
  String request = client.readStringUntil('\r');
  if (request.indexOf("/on") != -1) digitalWrite(ledPin, LOW);
  if (request.indexOf("/off") != -1) digitalWrite(ledPin, HIGH);
  // ... вывод HTML с кнопками
}
